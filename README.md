# yohan-profile-card

개인 프로필 카드 웹사이트 — 바이브코딩 프롬프트 라이브러리 실전 검증 프로젝트

## Stack

- Next.js 16 (App Router)

- TypeScript

- Tailwind CSS

- Vercel

## Features

- 프로필 영역 (이름 + 소개 + 아바타)

- 기술 스택 태그

- 프로젝트 카드 (제목 + 설명 + 링크)

- 다크 모드 기본

- 모바일 반응형

## Getting Started

npm install

npm run dev

# [http://localhost:3000](http://localhost:3000)
